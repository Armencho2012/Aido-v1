# Aido PCB

This folder contains the printed circuit board design files for the AIDO project.

## Files

- `MyFirstController.kicad_sch` — schematic design
- `MyFirstController.kicad_pcb` — PCB layout
- `MyFirstController.kicad_prl` — PCB rules file
- `MyFirstController.kicad_pro` — PCB project file
- `ERC.rpt` — electrical rules check report
- `images/` — board diagrams and reference images

## Notes

- Use KiCad to open and edit the PCB design files.
- Keep generated artifacts out of source control if you want the repository clean.
