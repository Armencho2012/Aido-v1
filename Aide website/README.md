# Aide website

This folder contains the production-ready frontend source for the AIDO project.

## Contents

- `src/` — main application source code
- `public/` — static assets and public files
- `package.json` — frontend dependency manifest and scripts
- `vite.config.ts` — Vite configuration for development and build
- `vercel.json` — deployment configuration for Vercel

## Notes

- This folder is for the website only; backend and firmware are in other folders.
- Keep only production source files here.
