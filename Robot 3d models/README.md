# Robot 3D Models

This folder contains the 3D model files for the AIDO robot design.

## Included files

- `Aido.stl` — main robot body for 3D printing or export
- `Aido.fbx` — main robot body for CAD, editing, or animation
- `back.stl` — secondary rear or cover part for printing
- `back.fbx` — secondary rear or cover part for CAD editing

## Usage

- Use `FBX` files for 3D modeling, editing, and visualization.
- Use `STL` files for 3D printing or slicing.
- Verify model units and orientation before exporting or printing.

## Notes

- Keep the `stl` and `fbx` files paired if you need both print-ready and editable versions.
